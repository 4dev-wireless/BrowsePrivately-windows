# BrowsePrivately-windows

This project creates a SOCKS5 proxy over SSH and launches a preconfigured portable browser that automatically uses it — essentially a simple DIY VPN-style client.

Initially, before opening TunnelBrowserLauncher.exe, run setup.bat and select "y" when prompted. 
Then select TunnelBrowserLauncher.exe. 
This should open a terminal and start plink.exe, it should also open a portable version of Firefox. 
Go to https://whatismyipaddress.com/ and check the ipv4 address shown. If it shows "" then it is working properly.
